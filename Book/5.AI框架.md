# AI 框架

## 一. 设计思路简介

1. AI 主要涉及的就是 NPC 和怪物的行为
2. 决策行为考虑使用行为树，但是为了能在子线程跑，以及防止跑完其他怪物的 AI 行为后影响到自己，所以需要在这一次 AI 的 Update 之前先搜集自己决策所需要的数据并缓存起来
3. 寻路相关行为直接用第三方库实现（DotRecast / ORCA）
4. 最终流程就是在每一次 AI 帧，先搜集数据即 **"知识"**，然后根据"知识"跑行为树即 **"决策"**，最后执行 **"决策结果"**

## 二. 三层架构

AI 框架采用 **知识（Knowledge）→ 决策（Decision）→ 控制（Control）** 三层架构：

```
AIComponent（每帧 Update）
  ├── 1. Knowledge（知识搜集）
  │     ├── 感知（Sensing）：视野、听觉
  │     ├── 目标（Target）：当前目标选定
  │     ├── 威胁（Threat）：威胁值管理
  │     ├── 技能（Skill）：可用技能查询
  │     ├── 战术（Tactic）：战术评估
  │     ├── 寻路（Path）：路径信息
  │     ├── 移动（Move）：移动状态
  │     ├── 区域（Area）：区域感知
  │     └── 行动（Action）：行动状态
  │
  ├── 2. Decision（决策）
  │     └── AIDecisionTree（决策树）
  │           ├── Root → Condition → Action
  │           └── DecisionArchetype（决策原型）
  │
  └── 3. Control（控制执行）
        ├── AIActionControl（行动控制）
        ├── AIBaseControl（基础控制）
        ├── AILocomotion（移动控制）
        │     ├── GoToTask（前往目标）
        │     ├── FollowMoveTask（跟随移动）
        │     ├── FacingMoveTask（朝向移动）
        │     ├── SurroundDashTask（环绕冲刺）
        │     ├── SnakelikeMoveTask（蛇形移动）
        │     └── RotationTask（旋转）
        └── MoveControl（移动模式）
              ├── Wander（游荡）
              ├── Surround（环绕）
              ├── Flee（逃跑）
              ├── MeleeCharge（近战冲锋）
              ├── PatrolFollow（巡逻跟随）
              ├── CombatFollow（战斗跟随）
              └── ...（约 20 种移动模式）
```

## 三. BrainModule 体系

AI 的各功能被拆分为独立的 BrainModule，位于 `Code/Game/Component/AI/BrainModule/`：

| 模块 | 说明 |
|------|------|
| `AISensingUpdater` | 感知更新（视野检测等） |
| `AITargetUpdater` | 目标更新（选择/切换目标） |
| `AIThreatUpdater` | 威胁值更新（仇恨管理） |
| `AISkillUpdater` | 技能更新（CD/可用性检查） |
| `AIPathfindingUpdater` | 寻路更新（路径计算） |
| `AIMoveUpdater` | 移动更新（位置同步） |
| `AIPoseControlUpdater` | 姿态更新（朝向/姿势） |

## 四. 编辑器

在本工程中通过 `Tools/Graph编辑器/AI编辑器` 打开，界面分为顶部菜单栏、左部节点编辑器、右部详情面板，编辑好后点击菜单栏中的导出。

**编辑器代码位置：** `Editor/DesignEditor/GraphEditor/AIEditor/`

| 文件 | 说明 |
|------|------|
| `AIGraphWindow.cs` | 编辑器窗口 |
| `AIGraph.cs` | Graph 数据 |
| `AIRootNode.cs` | 根节点 |
| `AIActionNode.cs` | 行为节点 |
| `AIConditionNode.cs` | 条件节点 |
